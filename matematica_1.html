1. Insiemi Numerici ed Espressioni (N, Z, Q)

Spiegazione Teorica Approfondita:
• N (Numeri Naturali): {0, 1, 2, 3...}. Chiuso rispetto ad addizione e moltiplicazione.
• Z (Numeri Interi Relativi): Introduce i numeri negativi per rendere possibile la sottrazione in ogni caso.
• Q (Numeri Razionali): Numeri esprimibili come frazione a/b (con b ≠ 0). Introduce le operazioni di divisione.
• Legge di Annullamento del Prodotto: Fondamentale per risolvere le equazioni scomposte. Afferma che se il prodotto di due o più fattori è zero, almeno uno dei fattori deve essere obbligatoriamente zero.
Esempi Svolti Passo-Passo: 1. Calcolo del M.C.D. e m.c.m. tra 24 e 36: Scomposizione: 24 = 2³ · 3 36 = 2² · 3² - M.C.D. (fattori comuni con minimo esponente): 2² · 3 = 12 - m.c.m. (fattori comuni e non comuni con massimo esponente): 2³ · 3² = 8 · 9 = 72 2. Applicazione della Legge di Annullamento del Prodotto: Risolvere: (x - 3)(2x + 4) = 0 Sotto-equazione 1: x - 3 = 0 ➔ x = 3 Sotto-equazione 2: 2x + 4 = 0 ➔ 2x = -4 ➔ x = -2 Soluzioni finali: x = 3 e x = -2

2. Calcolo Letterale: Monomi e Polinomi

Spiegazione Teorica Approfondita:
Un monomio è un'espressione letterale in cui compaiono solo moltiplicazioni e potenze tra numeri e lettere. Un polinomio è la somma algebrica di più monomi. I Prodotti Notevoli sono formule algebriche standard che permettono di calcolare rapidamente le potenze e le moltiplicazioni tra polinomi senza dover effettuare tutte le singole distribuzioni, riducendo drasticamente il rischio di errori di segno.
Formule dei Prodotti Notevoli ed Esempi Svolti: • Somma per Differenza: (A + B)(A - B) = A² - B² Esempio: (2x + 3y)(2x - 3y) = (2x)² - (3y)² = 4x² - 9y² • Quadrato di un Binomio: (A + B)² = A² + 2AB + B² *Attenzione al doppio prodotto!* Esempio: (3x - 4)² = (3x)² + 2(3x)(-4) + (-4)² = 9x² - 24x + 16 • Quadrato di un Trinomio: (A + B + C)² = A² + B² + C² + 2AB + 2AC + 2BC Esempio: (x + y - 2)² = x² + y² + 4 + 2xy - 4x - 4y • Cubo di un Binomio: (A + B)³ = A³ + 3A²B + 3AB² + B³ Esempio: (x + 2)³ = x³ + 3(x²)(2) + 3(x)(2²) + 2³ = x³ + 6x² + 12x + 8

3. Equazioni e Disequazioni di Primo Grado

Spiegazione Teorica Approfondita:
Un'equazione è un'uguaglianza tra due espressioni contenenti un'incognita. Si basa su due principi di equivalenza:
1. Primo Principio: Aggiungendo o sottraendo a entrambi i membri una stessa quantità, l'equazione resta equivalente.
2. Secondo Principio: Moltiplicando o dividendo entrambi i membri per una quantità diversa da zero, l'equazione resta equivalente.
Nelle Equazioni Fratte, l'incognita compare al denominatore. È obbligatorio porre le Condizioni di Esistenza (C.E.) prima di procedere, escludendo i valori che annullerebbero il denominatore (rendendo l'operazione impossibile).
Esempio di Equazione Fratta Interamente Svolta: Risolvere: 2 / (x - 1) = 4 / (x² - 1) Scomponiamo il denominatore di destra: x² - 1 = (x - 1)(x + 1) Fase 1: Condizioni di Esistenza (C.E.) x - 1 ≠ 0 ➔ x ≠ 1 x + 1 ≠ 0 ➔ x ≠ -1 Quindi: C.E. = {x ∈ R | x ≠ 1 e x ≠ -1} Fase 2: Calcolo del m.c.m dei denominatori m.c.m. = (x - 1)(x + 1) Moltiplichiamo entrambi i lati per il m.c.m per eliminare i denominatori: 2(x + 1) = 4 2x + 2 = 4 2x = 2 ➔ x = 1 Fase 3: Verifica delle soluzioni La soluzione trovata è x = 1. Ma per le C.E. avevamo stabilito che x ≠ 1. Conclusione: La soluzione NON è accettabile. L'equazione è IMPOSSIBILE.
